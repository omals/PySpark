# PySpark Learning Sessions

This repository contains a day-by-day PySpark learning track with practice notebooks, homework notebooks, and weekly reference documentation.

## Repository Overview

- `DAY 1.ipynb` to `DAY 12.ipynb`: Daily learning notebooks
- `DAY 2 HW.ipynb`, `DAY 3 HW.ipynb`, `DAY 4 HW.ipynb`, `DAY 6 HW.ipynb`, `DAY 8 HW.ipynb`, `DAY 9 HW.ipynb`, `DAY 10 HW.ipynb`, `DAY 11 HW.ipynb`, `DAY 12 HW.ipynb`: Homework notebooks
- `WEEK END TASK.ipynb`: End-of-week task notebook
- `documentation/week1.pdf`, `documentation/week2.pdf`, `documentation/week3.pdf`: Weekly documentation and references

## Learning Path

1. Start with the daily notebook for each day (`DAY X.ipynb`).
2. Complete the matching homework notebook where available (`DAY X HW.ipynb`).
3. Use files in `documentation/` to revise weekly concepts.
4. Attempt `WEEK END TASK.ipynb` after finishing the related week content.

## How to Open and Run

You can use one of the following:

- Databricks (recommended for Spark workflows)
- Jupyter Notebook / JupyterLab
- VS Code with the Jupyter extension

Typical local setup for Jupyter:

```bash
python -m venv .venv
source .venv/bin/activate
pip install pyspark notebook
jupyter notebook
```

## Notes

- Notebooks are organized in progressive order by day number.
- Homework files are available for selected days.
- Keep outputs and experiments versioned carefully when updating notebooks.
